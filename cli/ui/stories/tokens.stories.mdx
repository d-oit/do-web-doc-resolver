## Text Colors

| Token | Usage |
|-------|-------|
| `--wdr-text-primary` | Headings, body text |
| `--wdr-text-secondary` | Labels, captions |
| `--wdr-text-tertiary` | Placeholders, disabled |
| `--wdr-text-inverse` | Text on dark surfaces |
| `--wdr-text-link` | Hyperlinks |

## Typography

| Token | Value | Usage |
|-------|-------|-------|
| `--wdr-font-sans` | Inter, system-ui | UI text |
| `--wdr-font-mono` | JetBrains Mono, monospace | Code, data |
| `--wdr-text-xs` | 0.75rem | Badges, captions |
| `--wdr-text-sm` | 0.875rem | Body, inputs |
| `--wdr-text-md` | 1rem | Headings |
| `--wdr-text-lg` | 1.125rem | Section titles |

## Spacing Scale

| Token | px | Usage |
|-------|----|-------|
| `--wdr-space-1` | 4 | Inline gaps |
| `--wdr-space-2` | 8 | Tight padding |
| `--wdr-space-3` | 12 | Default padding |
| `--wdr-space-4` | 16 | Card padding |
| `--wdr-space-6` | 24 | Section gaps |
| `--wdr-space-8` | 32 | Layout margins |

## Border Radius

| Token | Value | Usage |
|-------|-------|-------|
| `--wdr-radius-sm` | 4px | Badges |
| `--wdr-radius-md` | 6px | Buttons, inputs |
| `--wdr-radius-lg` | 8px | Cards |
| `--wdr-radius-full` | 9999px | Pills, dots |

## Motion

| Token | Value | Usage |
|-------|-------|-------|
| `--wdr-duration-fast` | 100ms | Micro-interactions |
| `--wdr-duration-normal` | 200ms | Transitions |
| `--wdr-duration-slow` | 300ms | Layout changes |
| `--wdr-easing-default` | ease | General easing |

## Dark Mode

Toggle via Storybook toolbar. All tokens automatically invert through semantic layer — no component changes needed.
